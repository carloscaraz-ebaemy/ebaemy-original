# Kines

Sistema ERP de facturacion electronica, ecommerce, gestion de inventario y logistica.

## Stack

- **Backend:** Laravel (PHP)
- **Frontend:** Vue.js
- **Base de datos:** MySQL 8

## Modulos

- Facturacion electronica (SUNAT)
- Ecommerce (catalogo, carrito, checkout, pasarelas de pago)
- Smart Stock (stock fisico, alertas, movimientos)
- Logistico (guias de remision, transportistas)
- Promociones y descuentos
- Gestion de clientes y proveedores
- Reportes y dashboards
- Multi-tenancy

## Instalacion

1. Clonar el repositorio
2. `composer install`
3. `npm install && npm run dev`
4. Configurar `.env`
5. `php artisan migrate --seed`
